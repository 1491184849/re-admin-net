

# re-admin-net

前端使用re-admin，后端使用abp框架。rbac（基于角色权限访问控制）通用权限管理，精确到按钮级别的权限控制。

###### **使用步骤**

1. 数据库迁移

`dotnet migration database update`

2. 前端依赖下载

`yarn install`
